# Bike Frenzy

A Resale Bike Service
Where you can buy and sell used bike at resonable price.It is easy to sell and buy bikes on Bike Frenzy

## Features

- Login,Registration
- Add Product
- Buy Product
- Payment Method
- Report Product
- Verify user
- Jwt

## Built With

- FireBase Auth
- Vercel
- Google sign in
- MongoDb
- Jwt Token
